# Wiki Pages

This directory contains the Fro Bot knowledge wiki — persistent, compounding pages compiled from repository surveys, event interactions, and manual contributions.

See `knowledge/schema.md` for page conventions, frontmatter rules, and cross-reference syntax.

## Directory Structure

- `repos/` — per-repository knowledge pages
- `topics/` — cross-cutting technical topics
- `entities/` — named entities (tools, services, people, orgs)
- `comparisons/` — side-by-side analyses
